# Kyma-Miła

Installation guid

1. Clone repository
```
git clone https://github.com/pPrecel/Kyma-Mila
cd Kyma-Mila
```
2. Copy kiła.sh to bin 
```
cp kiła.sh /usr/local/bin/kiła
chmod +x /usr/local/bin/kiła
```
3. Remove repository
```
cd ..
rm -rf Kyma-Mila
```
4. Test your new favorite tool
```
kiła
```
